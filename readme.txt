
Juego de preguntas sobre economia Desarrollado en HTML5-CSS3-JavaScript(JQuery).

<h1>Caracteristicas:</h1>

Los estilos css son para entorno movil
Hay un temporizador y se tienen que responder preguntas de seleccion multiple o seleccion de imagen
Cada pregunta tiene un puntaje y una ayuda o pista
Solo podra usar las pistas en 3 preguntas despues se bloquea esta opcion
Si se acaba el tiempo se calificara automaticamente con las preguntas que tengan respuesta
